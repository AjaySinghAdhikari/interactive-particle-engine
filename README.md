
# ✋ Interactive Hand-Tracked Particle Engine

An interactive **3D particle visualization** controlled using **real-time hand tracking**.  
The project combines **Three.js** for high-performance WebGL rendering and **MediaPipe Hands** for gesture detection.

Users can **warp particles with hand movement** and **change particle shapes using hand claps**, creating an immersive visual experience.

---

## 🚀 Live Demo
👉 https://ajaysinghadhikari.github.io/interactive-particle-engine/

> ⚠️ Webcam permission is required for hand tracking.

---

## ✨ Features
- Real-time hand-tracked interaction
- 15,000 animated particles
- Smooth morphing between multiple procedural shapes
- Gesture-based shape switching (clap detection)
- Dynamic color changes based on hand position
- GPU-accelerated rendering using WebGL

---
